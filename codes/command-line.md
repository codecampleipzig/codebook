# Command Line

## Filepaths

|                 |                                                               |
| --------------- | ------------------------------------------------------------- |
| `/usr/bin/bash` | Absolute path: resolved relative to the file system root      |
| `img/logo.png`  | Relative path: resolved relative to current working directory |
| `.`             | Current working directory                                     |
| `..`            | Parent directory                                              |
| `~`             | User home directory                                           |
